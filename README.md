# Healthcare-Report-Dashboard
This Healthcare Report dashboard provides a comprehensive visualization of patient metrics, billing insights, and hospital trends using Power BI to drive data-informed healthcare decisions.

The Healthcare Analytics Dashboard is a comprehensive business intelligence solution designed to transform complex hospital data into actionable insights. By centralizing patient metrics, financial performance, and clinical trends, this tool enables healthcare administrators to monitor organizational health and improve operational efficiency through data-driven decision-making.

# Objectives

Operational Tracking: Analyzing yearly admission trends and Average Length of Stay (ALOS) to manage hospital capacity.

Financial Monitoring: Visualizing total and average treatment costs to optimize billing and revenue cycles.

Patient Profiling: Segmenting patient data by medical condition, admission type (Elective vs. Emergency), and insurance provider to tailor resource allocation.

Performance Benchmarking: Identifying top-performing hospitals and tracking physician-specific workloads.

# Tools Used

Microsoft Excel: The primary platform for data management and visualization.

Power Pivot & DAX: Used to create a robust Data Model and calculate complex Measures (like Average Treatment Cost) across multiple related tables.

PivotTables & PivotCharts: The engine used to aggregate 55,000+ patient records into digestible visual trends.

Advanced Formatting & Slicers: Implemented interactive slicers (Doctor, Gender, Hospital) to create a "software-like" user experience within a spreadsheet.

# Learnings

Data Modeling in Excel: Gained expertise in moving beyond simple tables to building a relational Data Model using Power Pivot for better performance with large datasets.

Automated Reporting: Learned how to build a "refreshable" architecture where new patient data can be added and reflected in the dashboard with a single click.

Dashboard UI/UX: Focused on creating a high-contrast, professional interface that mimics a web application, ensuring clarity for non-technical hospital staff.

Analytical Insight: Developed the ability to spot critical trends, such as the significant drop-off in 2024 admissions, which serves as a prompt for deeper root-cause analysis.
